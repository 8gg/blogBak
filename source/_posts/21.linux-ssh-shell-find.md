# 背景

知道一个IP，但是不知道它ssh端口开在哪里，怎么搞？

# 目前解法

先用masscan扫一下它的开放端口

```shell
masscan.exe 192.168.1.1 --ports 0-65535 --rate=10000
```

获取开放端口后，刚好windows10有个ssh命令

直接
```shell
ssh -p10000   192.168.1.1
```

通了就说明开了。

这种做法有点半手工，可能有更好的方法，但是目前没有搜索到。
