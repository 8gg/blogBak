# github是什么
每个人都会告诉你这是个代码托管平台。
但实际上我认为它还是个通过代码托管的一个泛社交平台。
可以和一些程序员名人互动，看看他们在搞什么程序。
所以玩github 是一定要注册账号的。

# 可以用github做什么

1.  代码托管
2.  个人学习经验的记录平台
3.  搭建博客
4.  认识各种人，左右观察
